DROP YOUR MUSIC SNIPPETS HERE
=============================

The hero "now playing" player reads its track list from:
    src/data/playlist.js

For each track, save a short .mp3 snippet in THIS folder using the
exact filename listed in that file. Current expected filenames:

    rocket-man.mp3        (Rocket Man — Elton John)
    headlines.mp3         (Headlines — Drake)
    back-in-black.mp3     (Back in Black — AC/DC)
    heavens-door.mp3      (Knockin' on Heaven's Door — Bob Dylan)
    get-lucky.mp3         (Get Lucky — Daft Punk)
    watchtower.mp3        (All Along the Watchtower — Jimi Hendrix)
    piano-man.mp3         (Piano Man — Billy Joel)
    beat-it.mp3           (Beat It — Michael Jackson)
    horse.mp3             (A Horse with No Name — America)

Tips
----
* Keep snippets short (15-30 seconds). Smaller files load fast and
  stay well within fair-use territory.
* .mp3 is the safest format for browser playback. .m4a / .ogg also
  work in most browsers.
* The player works before you add any files — it just stays silent
  and shows a small "drop the file" hint for missing tracks.
* DON'T commit full copyrighted songs to a public GitHub repo. Use
  short personal clips only.

To add a 10th track, uncomment the last entry in playlist.js, rename
it, and drop the matching file here.
